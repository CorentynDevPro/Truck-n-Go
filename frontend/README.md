# Frontend - Truck GPS (Vue 3 + Vite)

## Scripts

- `pnpm dev`: Start the development server
- `pnpm build`: Compile build for production
- `pnpm test`: Run tests (Vitest)
- `pnpm lint`: Check code style (ESLint)

## Convention

- Vue 3 Options API + TypeScript.
- ESLint + Prettier mandatory.
- Unit tests for each component and service.
